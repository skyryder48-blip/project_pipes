# Complete Weapon List by Category

This document lists all weapons organized by category with their naming conventions for stream folder organization.

## Naming Convention

```
WEAPON_[MODEL]         - Base weapon name (used in config)
[model]                - Stream folder name (lowercase, no WEAPON_ prefix)
```
