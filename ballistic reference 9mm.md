# Complete 9mm pistol ballistics reference for FiveM weapon meta

Full-size 9mm pistols cluster tightly around **1,100-1,200 fps** muzzle velocity with 115gr FMJ, with barrel length being the primary differentiator—every half-inch adds roughly **20-30 fps**. Weight ranges from the feather-light FN 509 at **763g empty** to the duty-ready Beretta M9 at **944g**, directly affecting recoil characteristics and handling speed. This reference provides exact specifications for 11 models across five manufacturers to calculate damage, bullet speed, recoil amplitude, and accuracy spread.

---

## Glock models share DNA but differ in critical handling characteristics

The Glock family uses identical **polygonal rifling** and **Safe Action** trigger systems, making them ideal candidates for baseline weapon balancing. Key variations stem from barrel length (affecting velocity/accuracy) and frame size (affecting capacity/recoil control).

|---------------|-------|-------|

| **124gr FMJ Velocity** | 1,090-1,130 fps | 1,095-1,135 fps |
| **Magazine Capacity** | 17 rounds | 17 rounds |
| **Weight Empty** | 22.05 oz / 625g | 22.05 oz / 625g |

| **Sight Radius** | 6.50" / 165mm | 6.50" / 165mm |
| **Overall Length** | 7.95" / 202mm | 7.95" / 202mm |

**Gen 5 improvements for game balancing:** The **Glock Marksman Barrel (GMB)** with enhanced rifling delivers measurably better accuracy, the lighter **26N trigger** (vs 28N) enables faster aimed fire, and the **flared magwell** reduces reload time. The **nDLC finish** offers no gameplay effect but justifies premium positioning.

**Unique handling characteristic:** The full-size grip counterbalances the lighter compact slide, creating **reduced muzzle flip** compared to a standard G19 despite identical barrel length. For game balancing, these should have **better recoil recovery** than the G19 but **identical velocity/damage**.

## Beretta pistols feature distinctive operating systems

Beretta's 92-series uses an **open-slide design** and **falling locking block** mechanism (Walther P38 derivative), while the PX4 Storm employs a **rotating barrel system** that fundamentally changes recoil characteristics.

### Beretta M9/92FS — Military-proven baseline

| Specification | Value |
|---------------|-------|
| **Barrel Length** | 4.9" / 125mm |
| **115gr FMJ Velocity** | 1,150-1,200 fps |
| **124gr FMJ Velocity** | 1,100-1,150 fps |
| **Magazine Capacity** | 15 rounds |
| **Weight Empty** | 33.3 oz / 944g |
| **Weight Loaded** | 40.8 oz / 1,157g |
| **DA Trigger Pull** | 10-12 lbs (first shot) |
| **SA Trigger Pull** | 4.5-5.5 lbs (subsequent) |
| **Sight Radius** | 6.1" / 155mm |
| **Overall Length** | 8.5" / 217mm |

**DA/SA mechanics for game balancing:** The **first shot** requires heavy trigger pressure (reduced first-shot accuracy), while subsequent SA shots are notably lighter. The heavy **944g empty weight** provides excellent recoil control but slower handling. The **open-slide design** virtually eliminates stovepipe malfunctions—highest reliability rating warranted.

### Beretta M9A3 — Modernized tactical variant

| Specification | Value |
|---------------|-------|
| **Barrel Length** | 5.1-5.2" / 130-132mm (threaded) |
| **115gr FMJ Velocity** | 1,180-1,220 fps |
| **124gr FMJ Velocity** | 1,120-1,170 fps |
| **Magazine Capacity** | 17 rounds |
| **Weight Empty** | 32-33.4 oz / 907-947g |
| **Weight Loaded** | ~43 oz / 1,219g |

**Improvements over M9:** The threaded barrel adds **20-30 fps** velocity and enables suppressor attachment, **Vertec-style grip** improves ergonomics, **17-round sand-resistant magazines** increase capacity, and **tritium night sights** aid low-light target acquisition. For game mechanics, this justifies: suppressor compatibility, +2 capacity, marginal accuracy boost.

### Beretta PX4 Storm — Rotating barrel recoil reduction

| Specification | Value |
|---------------|-------|
| **Barrel Length** | 4.0" / 102mm |
| **115gr FMJ Velocity** | 1,120-1,150 fps |
| **124gr FMJ Velocity** | 1,080-1,120 fps |
| **Magazine Capacity** | 17 rounds |
| **Weight Empty** | 27.7 oz / 785g |
| **Weight Loaded** | ~35 oz / 992g |

**Rotating barrel system explained:** When fired, the barrel rotates **45 degrees counter-clockwise** as it unlocks from the slide. This converts linear recoil energy into rotational movement, creating noticeably **softer felt recoil** and **reduced muzzle flip**. The **6-R rifling** generates counter-clockwise torque that assists the unlocking rotation.

**Game balancing implication:** Despite weighing **159g less** than the M9, the PX4 should have **equivalent or better recoil control** due to its rotating barrel. This is its defining characteristic—fastest follow-up shots of any Beretta despite being the lightest. Drawback: the rotating barrel causes suppressors to unscrew, so **no suppressor compatibility**.

| Specification | Value |

| **124gr FMJ Velocity** | 1,085-1,100 fps |

| **Weight Loaded** | ~37.5 oz / 1,063g |
| **Trigger Pull** | 5.4-6.5 lbs |
| **Trigger Reset** | 0.15-0.20" (short) |
| **Sight Radius** | ~6.3" / 160mm |
| **Overall Length** | 8.0" / 203mm |

**Modular Fire Control Unit:** The serialized steel chassis containing trigger, sear, and striker is removable and swappable between grip modules—a potential game mechanic for customization. The **M17/M18 military designation** (won XM17 competition in 2017) provides authenticity.

**Low bore axis** combined with substantial weight delivers **excellent recoil control**. The **short, crisp trigger reset** enables rapid aimed fire. For game balancing: high accuracy (longest barrel in category), excellent recoil control, moderate handling speed (heaviest modern striker-fired option).

## Recommended relative stat rankings for game balancing

**Damage (barrel length/velocity correlation):**
M9A3 > M9/92FS = G17 Gen5 = G17 Gen4 > P320 > G19X = G45 > TP9SF > G19 Gen4 = PX4 = FN 509

**Accuracy (sight radius, barrel quality, trigger):**
G17 Gen5 > M9A3 > P320 > G17 Gen4 > M9/92FS > G45 > G19X > TP9SF > PX4 > G19 Gen4 > FN 509

**Recoil control (weight, bore axis, operating system):**
M9A3 > M9/92FS > P320 > G17 Gen5 > G17 Gen4 > PX4 (rotating barrel bonus) > G19X > G45 > TP9SF > G19 Gen4 > FN 509

**Rate of fire (trigger weight, reset distance):**
TP9SF > G45 > G17 Gen5 > G19X > P320 > G17 Gen4 > G19 Gen4 > PX4 > FN 509 > M9 (DA first shot penalty)

**Handling/draw speed (weight, overall length):**
FN 509 > G19 Gen4 > PX4 > G45 > G19X > TP9SF > P320 > G17 Gen4 = G17 Gen5 > M9/92FS > M9A3

**Unique mechanics to implement:**
- **M9/M9A3:** DA/SA trigger—first shot less accurate, subsequent shots improved
- **PX4 Storm:** Rotating barrel—recoil reduction despite light weight, no suppressor
- **M9A3:** Threaded barrel—suppressor compatible
- **TP9SF:** Competition trigger—fastest aimed fire rate
- **P320:** Modular—potential customization mechanic
- **FN 509:** Lightweight—fastest handling, stamina-friendly
